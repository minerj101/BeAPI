Player

object

The player, quite an amazing creature eh?

Methods​

getName​

Get the players Microsoft account name.

getName();

Returns: string

getNameTag​

Get the players ingame name.

getNameTag();

Returns: string

getVanilla​

Gets the players vanilla player object.

getVanilla();

Returns: MCPlayer

getExecutableName​

Returns a valid name at which a command can be executed on.

getExecutableName();

Returns: string

setNameTag​

Sets the players ingame name.

setNameTag(name);
PARAMETERTYPEOPTIONALDEFAULTDESCRIPTION
namestring❌undefinedNew name.

Returns: void

getTags​

Returns all tags on player.

getTags();

Returns: string[]

hasTag​

Checks if player has tag.

hasTag(tag);
PARAMETERTYPEOPTIONALDEFAULTDESCRIPTION
tagstring❌undefinedTag name.

Returns: boolean

addTag​

Adds tag to player, returns true if player already has tag.

addTags(tag);
PARAMETERTYPEOPTIONALDEFAULTDESCRIPTION
tagstring❌undefinedTag name.

Returns: boolean

removeTag​

Removes tag from player, returns true if player did not have tag.

removeTag(tag);
PARAMETERTYPEOPTIONALDEFAULTDESCRIPTION
tagstring❌undefinedTag name.

Returns: boolean

getGamemode​

Gets the players gamemode.

getGamemode();

Returns: Gamemodes

getLocation​

Gets the players location.

getLocation();

Returns: Location

sendMessage​

Sends a message to player.

sendMessage(message);
PARAMETERTYPEOPTIONALDEFAULTDESCRIPTION
messagestring❌undefinedMessage body to send user in chat.

getInventory​

Gets the players inventory.

getInventory();

Returns: Inventory

executeCommand​

Executes command as the player.

executeCommand(command, debug);
PARAMETERTYPEOPTIONALDEFAULTDESCRIPTION
commandstring❌undefinedCommand to execute.
debug?boolean✔️falsePipe debug info.

Gets score value on certain objective of player.

PARAMETERTYPEOPTIONALDEFAULTDESCRIPTION

getHealth();

Returns: Health

Checks if player is within a zone.

inPosition(x, y, z, dx, dy, dz);
PARAMETERTYPEOPTIONALDEFAULTDESCRIPTION
xnumber❌nullX coordinate.
ynumber❌nullY coordinate.
znumber❌nullZ coordinate.
dxnumber❌nullDX coordinate.
dynumber❌nullDY coordinate.
